# Palette Journal

## 2025-02-24 - Context Menu Scannability
**Learning:** Adding icons to context menus significantly improves scannability, especially for destructive actions like "Delete" or "Empty". Users can identify actions by shape/color before reading the text.
**Action:** Always consider adding `leadingIcon` to `DropdownMenuItem` components in Jetpack Compose to align with Material Design guidelines and improve UX.
