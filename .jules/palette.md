## Palette Journal
## 2024-05-22 - [Accessibility: Content Descriptions for Visual Counters]
**Learning:** Custom visual components like counters often lack inherent semantic meaning for screen readers. Merging descendants and providing a constructed content description (e.g., "Total New: 10") is crucial for non-visual users to understand what the number represents.
**Action:** Always verify that custom visual indicators have a corresponding content description that explains both the value and its label.
